# Webtigo - Premium Web Tools

A powerful, serverless-ready web application built with:
- **Node.js 20.x** & **Express**
- **EJS** for Server-Side Rendering
- **Sharp** for high-performance image processing
- **PDF-Lib** for PDF manipulation (Pure Node.js)

## Features
- PDF Tools (Split, Create)
- Image Compressor & Resizer
- QR Code Generator
- Text-to-Speech
- Frequency Generator

## Deployment
Deployed on Vercel. 
*Note: Ensure "Node.js Version" is set to 20.x in Vercel Settings.*
